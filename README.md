# Конвертер учебного конфигурационного языка в XML

Этот инструмент преобразует текст из учебного конфигурационного языка в XML формат.
Вариант команды для запуска через командную строку в папке с файлами: python test_parser.py
## Особенности языка

### Комментарии
- Однострочные: `// комментарий`
- Многострочные: `{- комментарий -}`

### Типы данных
- **Числа**: целые, дробные, экспоненциальная форма
  - `42`, `-3.14`, `1.5e2`
- **Строки**: в одинарных кавычках
  - `'текст'`, `'It\'s great'`
- **Массивы**: в круглых скобках через запятую
  - `(1, 2, 3)`, `('a', 'b', 'c')`
